# Data-Preprocessing-Cleaning
This is full python project for Data Preprocessing &amp; Cleaning. If you need then you can use it. you can make your data Lower case, HTML tags remove, url remove, punctuation handling, Chat conversion handling, incorrect text handling, remove imoji handling, stopwords and tokenizing.

Data link: https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews
